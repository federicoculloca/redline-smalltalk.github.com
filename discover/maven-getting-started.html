Maven and Redline Smalltalk

Currently, we only have a Redline Smalltalk snapshot meaning you need to specify the Sonatype snapshot repository in your section: {% highlight bash %} snapshots-repo https://oss.sonatype.org/content/repositories/snapshots false true {% endhighlight %}

Maven Dependency

Add the following stanza to your to include Redline Smalltalk in your projects. {% highlight bash %} st.redline redline 0.3-SNAPSHOT {% endhighlight %}

You are now ready to work with Redline.
